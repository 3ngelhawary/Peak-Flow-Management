Concept

Retention principle

The tank stores the temporary excess when storm inflow is higher than the pump station discharge capacity.

V = (Qin − Qpump) × T × SF

Simulation logic

The synthetic mode estimates peak runoff using the Rational Method, then creates a triangular hydrograph for storage routing.

The live mode converts forecast precipitation into an hourly inflow series using the same catchment area and runoff coefficient.
